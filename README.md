# agastya-portfolio

CV-first portfolio website built with React + Vite and prepared for GitHub Pages.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## GitHub Pages deployment

This project is configured with:

- `vite.config.js` base: `/agastya-portfolio/`
- deploy scripts in `package.json`

Deploy with:

```bash
npm run deploy
```
